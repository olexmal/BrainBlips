BrainBlips Documentation

This folder contains documentation for the BrainBlips project setup and configuration.

Contents​

  • Cursor Setup Guide - Complete guide to the Cursor IDE configuration for note-taking and research
  • Project Structure - Overview of the project organization
  • MCP Servers Configuration - Model Context Protocol servers setup

Quick Start​

  1. Read the Cursor Setup Guide to understand the complete configuration
  2. Review the Project Structure to understand file organization
  3. Check MCP Servers Configuration for advanced features

Project Overview​

BrainBlips is a knowledge management and research workspace configured with:

  • Cursor IDE as the primary development environment
  • Obsidian integration for note-taking and knowledge graphs
  • MCP Servers for enhanced AI capabilities
  • Docker support for containerized development
  • Git integration for version control
